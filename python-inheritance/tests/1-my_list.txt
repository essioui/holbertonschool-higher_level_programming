Module MyList
=============

Usage
-----

import function from Module:
    >>> MyList = __import__('1-my_list').MyList

Create an instance of MyList:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

test add elements of list:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> print(my_list)
    [1, 2, 3]

test print list sorted:
    >>> my_list.sorted()
    [1, 2, 3]

test replace with index:
    >>> my_list[0] = 7
    >>> print(my_list())
    [7, 2, 3]

test remove element:
    >>> my_list[0] = 5
    >>> print(my_list)
    [5, 2, 3]
